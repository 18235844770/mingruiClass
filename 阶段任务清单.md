# 教育培训机构后台管理系统｜阶段任务清单（执行版）

## 使用说明
- 本清单按阶段顺序执行，不跳阶段。
- 每个阶段必须完成「任务项 + 输出物 + 验收标准」后再进入下一阶段。
- 核心红线：不改业务规则、校区隔离优先、课时和金额数据准确、关键操作可追溯。
