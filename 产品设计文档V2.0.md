# 教育培训机构后台管理系统 — 产品介绍文档

## 1️⃣ 产品概述

**产品名称**：教育培训机构后台管理系统

**产品类型**：企业内部管理系统

**目标用户**：

-  **管理员**：管理系统用户、角色、校区，查看报表和操作日志 
-  **销售**：管理学生信息、课程录入、消课和低课时提醒 
-  **老板**：查看业务仪表盘、报表、操作日志 

**产品目标**：

1.  清晰管理学生、课程和课时余额 
2.  支持多角色、多校区管理，保证数据隔离 
3.  提供实时业务监控和报表导出 
4.  支持实际业务灵活性：课程价格可不一致、超额消课、欠费上课 

------

## 2️⃣ 核心功能模块与详细说明

### 2.1 学生管理

**功能点**：

- 添加学生
  -  输入姓名（校区内唯一）、手机号（可选）、总金额、是否交费、备注信息 
  -  校区选择（如果销售负责多个校区，登录后选择校区） 
  -  弹窗添加课程信息：课程名称、价格、课时、类型（1v1/班课）、备注 
-  **编辑/删除学生及课程信息** 
- 查询与筛选
  -  学生列表可按姓名、剩余课时、欠费状态、课程类型筛选 
  -  显示剩余课时、总课时、课程价格 
- 低课时提醒
  -  剩余课时 < 3 学生自动标记 
  -  销售首页固定显示低课时提醒列表，点击可直接进入消课界面 

**特点**：

-  允许课程价格与总金额不一致，符合实际场景 
-  校区内姓名唯一校验，防止重复录入 
-  无班级容量限制，只记录课时 

------

### 2.2 消课管理

**功能点**：

- 消课操作
  -  选择学生 → 选择课程 → 输入消课课时 → 消课时间自动记录 → 更新剩余课时 
  -  剩余课时可为负数，高亮显示 
- 消课记录查询
  -  可按学生、课程、剩余课时、消课日期筛选 
  -  列表显示消课时间，避免重复消课 
- 欠费消课
  -  欠费学生仍允许消课 
  -  界面高亮欠费状态，提醒销售 

**特点**：

-  超额消课允许，符合实际业务场景 
-  消课记录精确到时间，保证数据可追溯 

------

### 2.3 校区与权限管理

**功能点**：

- 校区管理

  （管理员） 

  -  创建、编辑、删除、查看校区 
  -  销售账号绑定校区 

- 校区隔离

  -  销售登录后选择校区 
  -  所有学生、课程、消课、报表数据基于校区显示 

- 角色管理

  （管理员） 

  -  创建、编辑、删除、查看角色 
  -  分配权限：用户管理、学生管理、消课管理、报表管理、日志管理 

**特点**：

-  数据隔离保证不同校区信息安全 
-  多校区销售支持，一个销售可管理多个校区 
-  日志记录操作行为，追踪校区切换和关键操作 

------

### 2.4 仪表盘

**功能点**（老板可查看）：

-  在读学生人数 
-  课程总课时 
-  剩余课时 
-  总收入 
-  未收入课时费用 
-  按校区统计，支持时间筛选 

**特点**：

-  首页固定低课时提醒 
-  可快速看到校区整体运营状态 

------

### 2.5 报表管理

**功能点**（管理员、老板可查看）：

-  数据报表展示：学生人数、课程总课时、剩余课时、总收入、未收入课时 
-  支持按校区、销售、课程类型、日期区间筛选 
-  支持导出 Excel/PDF 

**特点**：

-  数据按校区汇总 
-  报表统计基于实际录入价格和付款状态 
-  帮助老板快速做财务和业务分析 

------

### 2.6 日志管理

**功能点**（管理员、老板可查看）：

-  查看操作日志 
-  删除日志 
-  日志按用户、操作类型、时间筛选 

**特点**：

-  关键操作可追溯（如消课、学生删除、校区切换） 
-  提高系统安全性和数据可控性 

------

## 3️⃣ 核心业务流程

1. 销售登录与校区选择
   -  销售账号由管理员创建 
   -  登录后选择校区 → 系统加载该校区数据 
2. 学生录入流程
   -  学生添加 → 弹窗录入课程信息 → 系统生成 `student` 和 `student_course` 记录 
3. 消课流程
   -  选择学生 → 选择课程 → 输入消课课时 → 系统更新剩余课时 → 生成 `course_consumption` 记录 
   -  剩余课时 < 3 → 标记低课时 → 首页提醒 
   -  欠费学生可继续消课，界面高亮 
4. 报表与仪表盘流程
   -  系统统计学生人数、总课时、剩余课时、总收入、未收入课时 
   -  按校区、销售、课程类型筛选 → 支持导出 
5. 管理员操作流程
   -  用户/角色/校区管理 → 权限分配 → 日志管理 → 报表查看 

## 4️⃣ 角色权限说明

| 角色   | 可访问模块                               | 功能点                                         |
| ------ | ---------------------------------------- | ---------------------------------------------- |
| 管理员 | 用户管理、角色管理、校区管理、报表、日志 | CRUD用户/角色/校区，查看报表，操作日志管理     |
| 销售   | 学生管理、消课、低课时提醒               | 添加/编辑/删除学生及课程，消课，查看低课时提醒 |
| 老板   | 仪表盘、报表、日志                       | 查看仪表盘、报表，导出报表，操作日志查看       |
