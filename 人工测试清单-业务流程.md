- [ ] 启动后端、前端服务

## 1. 登录与权限
- [ ] `admin` 登录成功，进入管理员菜单

- [ ] 新建角色成功
- [ ] 重复 `code` 创建失败并提示

### 2.3 用户管理
- [ ] 新建用户成功（角色/状态/校区）
- [ ] 编辑用户成功（角色/状态/重置密码）
- [ ] 删除普通用户成功

### 2.4 日志管理
- [ ] 日志列表筛选可用（模块/操作人/时间）
