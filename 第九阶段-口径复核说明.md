# 第九阶段口径复核说明（样本 SQL）

> 目的：为第九阶段提供“可复核”依据，便于核对仪表盘、报表与导出结果一致性。

---

## 1. 统一筛选参数约定

联调时请先固定一组筛选参数（示例）：

- `campus_id = :campusId`（可为空，空表示全量）
- `start_time = :startTime`，`end_time = :endTime`（可为空）
- `sales_id = :salesId`（可为空）
- `course_type = :courseType`（可为空）

说明：

- 学生与课程视图的时间过滤默认按 `created_at`
- 消课视图时间过滤按 `consumption_time`
- 所有统计均以软删口径过滤（`deleted_at IS NULL`）

---

## 2. 仪表盘 5 个核心指标复核 SQL

### 2.1 在读人数

```sql
SELECT COUNT(1) AS active_students
FROM students s
WHERE s.deleted_at IS NULL
  AND (:campusId IS NULL OR s.campus_id = :campusId)
  AND (:salesId IS NULL OR s.created_by = :salesId)
  AND (:startTime IS NULL OR s.created_at >= :startTime)
  AND (:endTime IS NULL OR s.created_at <= :endTime);
```

### 2.2 总收入

```sql
SELECT COALESCE(SUM(s.total_amount), 0) AS total_income
FROM students s
WHERE s.deleted_at IS NULL
  AND (:campusId IS NULL OR s.campus_id = :campusId)
  AND (:salesId IS NULL OR s.created_by = :salesId)
  AND (:startTime IS NULL OR s.created_at >= :startTime)
  AND (:endTime IS NULL OR s.created_at <= :endTime);
```

### 2.3 总课时 / 剩余课时

```sql
SELECT
  COALESCE(SUM(sc.total_hours), 0) AS total_course_hours,
  COALESCE(SUM(sc.remaining_hours), 0) AS total_remaining_hours
FROM student_courses sc
JOIN students s ON s.id = sc.student_id
WHERE sc.deleted_at IS NULL
  AND s.deleted_at IS NULL
  AND (:campusId IS NULL OR s.campus_id = :campusId)
  AND (:salesId IS NULL OR s.created_by = :salesId)
  AND (:courseType IS NULL OR sc.course_type = :courseType)
  AND (:startTime IS NULL OR sc.created_at >= :startTime)
  AND (:endTime IS NULL OR sc.created_at <= :endTime);
```

### 2.4 未收入课时费用（average 示例）

```sql
SELECT COALESCE(SUM(
  CASE
    WHEN sc.total_hours = 0 THEN 0
    ELSE sc.remaining_hours * (sc.course_price / sc.total_hours)
  END
), 0) AS unearned_course_fee
FROM student_courses sc
JOIN students s ON s.id = sc.student_id
WHERE sc.deleted_at IS NULL
  AND s.deleted_at IS NULL
  AND (:campusId IS NULL OR s.campus_id = :campusId)
  AND (:salesId IS NULL OR s.created_by = :salesId)
  AND (:courseType IS NULL OR sc.course_type = :courseType)
  AND (:startTime IS NULL OR sc.created_at >= :startTime)
  AND (:endTime IS NULL OR sc.created_at <= :endTime);
```

## 3. 报表视图复核重点

### 3.1 学生视图

- 核对字段：`studentName`、`totalIncome`、`totalCourseHours`、`totalRemainingHours`
- 核对方式：抽样 3 名学生，对比学生详情页课程汇总

### 3.2 课程视图

- 核对字段：`totalHours`、`remainingHours`、`consumedHours`
- 核对方式：确认 `consumedHours = totalHours - remainingHours`

### 3.3 消课视图

- 核对字段：`consumptionTime`、`consumedHours`、`remainingHoursAfter`
- 核对方式：抽样对照 `course_consumptions` 明细记录

## 4. 导出一致性核对

同一筛选参数下，核对以下一致：

- 仪表盘汇总值
- 报表页面汇总值
- Excel 汇总页指标
- PDF 汇总区指标

建议记录：

- 查询时间
- 查询参数
- 页面截图
- 导出文件名
- 差异说明（若有）
